<h1>This is a django backend for the Farm-Assist Application.</h1>

<h3>To run the backend :</h3>

<ol>
<li>Clone the repository.</li>
<li>Create a virtual enviornment using <em>venv</em><br>
  
    > python -m venv venv
</li>
<li>Activate the virtual enviornment.

    > .venv\Scripts\activate
</li>
<li>Install the dependencies.
  
    > pip install requirements.txt
</li>
</ol>

<h3>Finally run the project.</h3>

```shell
> python manage.py runserver
```
